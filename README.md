# 🚗 New Wheels Business Analysis

## 📌 Project Overview

This project analyzes the business performance of a used vehicle resale company using SQL.

The objective was to identify revenue trends, customer satisfaction issues, and operational bottlenecks through SQL-based business analysis.

---

## 🛠 Tools Used

- MySQL
- MySQL Workbench

---

## 📚 SQL Concepts Used

- SELECT
- WHERE
- GROUP BY
- HAVING
- INNER JOIN
- LEFT JOIN
- CASE WHEN
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- RANK()
- DENSE_RANK()
- ROW_NUMBER()

---

## 📊 Business Questions Solved

- Quarterly revenue analysis
- Customer rating trends
- Shipping delay analysis
- Top-performing vehicle brands
- Customer purchase behaviour
- Revenue by state
- Top customers
- Monthly sales trend

---

## 📈 Key Insights

- Revenue declined across multiple quarters.
- Shipping delays significantly impacted customer ratings.
- Customer satisfaction decreased due to longer delivery times.
- Identified top-performing regions and products.
